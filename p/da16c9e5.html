【Python】利用 Python+pyexif 库获取以及修改照片 EXIF 信息

发表于2021-04-11|更新于2021-07-12|Python
|字数总计:1.3k|阅读时长:5分钟|阅读量:

1. 背景

  在《【Python】基于 Windows 平台批量修改图片 EXIF 信息》这篇博客中，简单介绍了下在 Windows 平台，如何利用 python 对照片进行修改时间等 exif 信息，其中用到的第三方库是 win32file 和 pywintypes，这两个库只能在 Windows 平台中使用，换到 macOS 或 Linux 平台，就无法使用了，具有一定的局限性。
  本篇博客，则基于 macOS 平台，通过利用 pyexif 库，重新编写代码，实现修改照片时间信息的功能；同时，还顺带介绍下如何获取照片中的 GPS 信息，并对其进行修改操作。

2. 环境准备

序号 名称 备注
1 macOS 我的是 macOS High Sierra 系统
2 Python 3.X 版本以上
3 piexif 第三方库，通过 pip install piexif 命令安装即可
4 PIL 第三方库，通过 pip install pillow 命令安装即可
