|字数总计:1.4k|阅读时长:6分钟|阅读量:

1. 背景

  在《【Python】基于Windows平台批量修改图片EXIF信息》这篇博客中，简单介绍了下在Windows平台，如何利用python对照片进行修改时间等exif信息，其中用到的第三方库是win32file和pywintypes，这两个库只能在Windows平台中使用，换到macOS或Linux平台，就无法使用了，具有一定的局限性。
  本篇博客，则基于macOS平台，通过利用pyexif库，重新编写代码，实现修改照片时间信息的功能；同时，还顺带介绍下如何获取照片中的GPS信息，并对其进行修改操作。

2. 环境准备

1 macOS 我的是macOS High Sierra系统
2 Python 3.X版本以上
3 piexif 第三方库，通过pip install piexif命令安装即可
4 PIL 第三方库，通过pip install pillow命令安装即可
