1. 背景

  由于树莓派的 CPU 是 ARM 架构，在使用 docker 拉取 MySQL 数据库镜像时，会提示找不到与 linux/arm64/v8 匹配的镜像。而 MariaDB 数据库是属于 MySQL 的一个分支，完全兼容 MySQL，所以可以成为 MySQL 的替代品。   本篇博客，将记录如何在 arm64 位 Ubuntu 系统的树莓派上，基于 docker 容器部署 MariaDB 数据库。
