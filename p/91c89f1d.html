【Linux】树莓派 arm64 位 Ubuntu 系统基于 docker 安装 MariaDB 数据库

发表于2021-03-14|更新于2021-06-20|Linux
|字数总计:1.2k|阅读时长:5分钟|阅读量:

1. 背景

  由于树莓派的 CPU 是 ARM 架构，在使用 docker 拉取 MySQL 数据库镜像时，会提示找不到与 linux/arm64/v8 匹配的镜像。而 MariaDB 数据库是属于 MySQL 的一个分支，完全兼容 MySQL，所以可以成为 MySQL 的替代品。
  本篇博客，将记录如何在 arm64 位 Ubuntu 系统的树莓派上，基于 docker 容器部署 MariaDB 数据库。

2. 拉取镜像

  执行以下步骤之前，确认树莓派上已安装好 docker，并正常运行。可参考《【Linux】基于树莓派 4B 安装 Ubuntu Server 系统及必要配置》第 8 章节。

  1. 执行 sudo apt update 命令，更新缓存。
  2. 执行 docker search mariadb 命令，搜索镜像。
  3. 执行 docker pull mariadb 命令，拉取镜像。
  4. 执行 docker images 命令，查看镜像信息，记录 images ID。
